# Security Policy

Swift Quiz Academy is an educational iOS project and does not currently handle accounts, payments, or server-side user data.

## Reporting a Vulnerability

If you find a security issue, please do not open a public issue with exploit details.

Instead, contact the maintainer through GitHub:

https://github.com/BobbyUzunov

Please include:

- A clear description of the issue.
- Steps to reproduce it.
- The affected app version or commit.
- Any suggested fix, if available.

## Supported Versions

Security fixes are considered for the latest version on the `main` branch.
